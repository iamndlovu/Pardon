about me

I'm a self-taught front-end web developer who enjoys solving problems with Vanilla JavaScript, React.js and vs code.

I started learning web development in 2017 soon after compliting high school. I've been freelancing since 2019, and have built web apps for both small companies and individuals. I also learnt other programming languages like C++ and Java in University.

front-end web developer?

As a front-end developer I am responsible for developing the part of a web app that the user sees and interacts with. In simple terms, my job is to make sure no one feels like hanging themselves every time they open the website.

I develop responsive websites that look great and interact flawlessly with the user on all devices. I also try to include as much accessibility features as possible on all apps to ensure that even those using screen readers can have a great user experience.

To achieve this task I use HTML, CSS, Sass, DOM, JavaScript, React, and other tools and technologies. I use a headless CMS, Sanity, when a project requires a backend.
